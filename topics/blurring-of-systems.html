Blurring of Systems

What is MCP?

Model Context Protocol (MCP) is an open-source communication standard that allows LLMs to connect and interoperate with external tools, data sources, and systems.

Focus on Workflows

With more external systems supporting MCP for use by AI models and agents, people's attention shifts from knowing how to do "x,y,z" in a integrated library system to doing task 1, task 2, and task 3 that spans multiple systems with primary interface being a text prompt to an AI Agent.

Even without a MCP endpoint, AI Agents can manipulate web browsers and native apps to accomplish goals coordinated by the library users and not as much on the learning the specifics of a various library system platforms, digital repositories, electronic databases, and websites.

Blue Core & FOLIO

Blue Core is a collaborative library metadata environment of linked open BIBFRAME data, cooperatively managed by multiple institutions, that share BIBFRAME Works and Instances, for downstream library uses like discovery, circulations, and acquisitions. Blue Core attempts to reduces or eliminates duplicative workflows among the member institutions.

Of the initial set of five partners, three (Library of Congress, Stanford University, and Cornell University) are all FOLIO libraries. From its inception, Blue Core has been "AI forward" with the prototype's API also providing MCP endpoints for models and AI Agents.
