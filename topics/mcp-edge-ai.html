MCP for Edge AI

Interface for Models & AI Agents

In the current implementation of the FOLIO edge-ai module, the API endpoints are also available for use by LLMs and AI Agents by supporting MCP.

A model or agent environment like Antropic's Claude Code or OpenAI's Codex can, using the MCP, interact with an internal Inventory agent to create a FOLIO Inventory Instance with either a text prompt or by uploading a cover image.
