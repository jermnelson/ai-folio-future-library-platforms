A Short History of Edge AI

Date Description
16 November 2023 Code for the presentation Chatting with your Catalog: Exploring the use of LLMs with FOLIO LSP for the 2023 Fantastic Futures conference, was the bases for the first iteration of the edge-ai.
3 June 2024 First commit in edge-ai module in Github.
23 September 2024 Edge-AI used as part of the WOLFcon 2024 Workshop, Understanding and Using AI Workflows with FOLIO, in London.
8 April 2025 Major refactor of code to use Pydanic AI with AI Agents.
28 April 2025 Edge-AI deployed for Stanford University Libraries use in internal web app.
18 September 2025 Edge-AI support for Model Context Protocal (MCP)
23 September 2025 Edge-AI critical component of the WOLFcon 2025 Workshop, Using Generative AI with FOLIO, in Kansas City.
