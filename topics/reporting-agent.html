Install AI Coding Assistants

  • Download and install Claude Code: curl -fsSL https://claude.ai/install.sh | bash or with Windows Powershell irm https://claude.ai/install.ps1 | iex
  • Download and install Gemini CLI: npm install -g @google/gemini-cli
  • Download and install Open AI Codex CLI: npm i -g @openai/codex
