# AI + FOLIO: Future of Library Platforms

## Abstract
As libraries adopt open-source platforms like FOLIO, the integration of AI 
offers new possibilities for automation and smarter workflows. Jeremy will
demonstrate how AI can enhance acquisitions, resource management, and 
API-driven processes within FOLIO, creating efficiencies that reduce staff 
workload. The session will highlight real-world examples and future aspirations 
for AI-enabled library systems, including predictive analytics and automated 
decision-making. Participants will gain actionable ideas for leveraging AI 
in their own FOLIO environments.

Presentation located at https://jermnelson.github.io/ai-folio-future-library-platforms/
